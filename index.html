EVENT 2- Code0niX (CODING COMPETITION)



Date of Event : 23rd April 2022; Saturday
Location : Computer Lab1/Computer Lab2
Event Type : Technical,Hybrid
Duration : 4 hours(3-hours main event +1 hour to conclude)
Sponsors : -
Coordinators : Anuj Singh Negi


  • An Algorithm-Based/Coding Event where participants from VIT Bhopal can show off their coding skills, learn some new paths in the field of coding and earn too.
  • This event is a 3-hour straight coding event. Questions will be designed by the organizing team. Questions will based on the theme of Club ANTERiX.
  • This is a hybrid-mode event. Participants can attend it via offline or online mode. We'll focus to majorly shift it to offline mode for better discipline or code of conduct.
  • This is an individual event. Competition will be strict toward plagiarism check.
  • Preferred Platforms - Codechef or Dare2Compete. (Participants have to Sign In/Sign Up using VIT Bhopal mail ID)
  • Default Rating system of the above mentioned platform will be used to Rank the Participants. (Default Priority - Correct Questions, % of Test Cases Passed, Time Taken)
  • Participants have to register prior the commencement of the event. No Registration Fees required.
  • Top 3 ranks and Top 2 rank holder from Club ANTERiX will be awarded rewards. Rank 1 will be declared as the Winner of the Event.
  • Volunteers are invited to join in but cannot participate in the event.
  • Participatation, Winning and Volunteering Certificates will be provided.
